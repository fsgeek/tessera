# Project Tessera

The purpose of this project is to take an idea on which I have been building
and mapping it into cloud native services via a structured process where I
use AI tools to (hopefully) simplify the journey.

To support this I'm going to be tracking the work via a series of journal entries.

The repository will require:
1. My own GPG key, so I am the one signing the commits.
2. Signed code commits (enforced on github)
3. OTS timestamps on the commmits
4. A public record of my work
